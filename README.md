# Investor 系统
## 目前的功能
----

- 爬取网贷系统 （Octopus）
- 实时显示数据  （eagle）

## 依赖
---

- python 2.7+
- scrapy >= 0.24.4
- redis-py >= 2.10.3
- socket.io 
- node_redis
- express
- angularjs 1.3.15
- bootstrap 3

## 安装

在根目录下运行下面的命令安装python的依赖包:

    pip install -r requirements.txt
    
在eagle目录下运行下面的命令安装JS依赖包:

    npm install
    bower install angularjs
